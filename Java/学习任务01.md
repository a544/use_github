# Java学习任务01
## 目标
用户登录
## 功能描述
输入用户名和口令, 验证是否正确, 正确则显示登录成功; 不正确显示登录失败.
## 技术要求
### 界面
采用命令行方式实现
### 数据存储
直接在程序的变量中保存正确的用户名和口令

# Java学习任务09
## 目标
用户登录
## 功能描述
输入用户名和口令, 验证是否正确, 正确则显示登录成功; 不正确显示登录失败.
登录成功后增加一些进一步填写用户信息的功能.
## 技术要求
### 界面
网页界面方式实现, 采用HTML5和一种流行的前端框架
### 服务器端
采用Spring MVC实现. 采用第三方认证程序进行身份认证. 
### 数据存储
增加kv的缓存数据库. 关系数据库. 采用JDBC template访问数据库.
